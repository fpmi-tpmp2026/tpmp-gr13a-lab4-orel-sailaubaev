# Ипподром
### Описание:
Проект представляет собой консольную информационно-управляющую систему для автоматизации работы ипподрома и проведения соревнований. Приложение позволяет вести структурированный учет лошадей, их владельцев, штатных жокеев и детально фиксировать результаты забегов. Система поддерживает ролевую модель доступа: администраторы могут управлять базой данных и автоматически рассчитывать распределение призового фонда, в то время как владельцы и жокеи получают доступ к своей личной статистике. Главной особенностью функционала является генерация аналитических отчетов, позволяющих выявлять самых успешных лошадей и наиболее активных участников соревнований.
### Установка:
Для локальной установки и сборки приложения вам потребуется компилятор C/C++ (например, GCC или Clang), система сборки (make или cmake), а также установленная библиотека SQLite3.
1. Склонируйте репозиторий с исходным кодом на ваш локальный компьютер:
   ```Bash
   git clone https://github.com/ваша-ссылка-на-репозиторий.git
2. Перейдите в корневую директорию проекта:
   ```Bash
   cd name-of-repo
3. Выполните сборку проекта. Если используется стандартный Makefile, введите:
   ```Bash
   make
(Если вы используете CMake, создайте папку сборки: mkdir build && cd build && cmake .. && make)После успешной сборки исполняемый файл приложения будет помещен в директорию bin.
### Использование:
Для начала работы с приложением выполните следующие шаги:
1. Запустите скомпилированный файл из терминала, находясь в корневой директории проекта:
  ```Bash
  ./bin/hippodrome
```
2. При запуске система предложит вам пройти аутентификацию. Введите свой логин и пароль.
В зависимости от ваших учетных данных, система автоматически определит вашу роль (Администратор, Жокей или Владелец лошади) и предоставит соответствующее меню.
3. Используйте цифры на клавиатуре для навигации по меню консоли. Например, Администратор может выбрать пункт "Распределить призовой фонд", а Жокей — "Просмотреть мои забеги".
4. Для безопасного выхода из системы выберите пункт "Выход" в главном меню.
